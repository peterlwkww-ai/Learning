# 💰 Daily Expense Tracker

A simple, fast, and mobile-friendly expense tracker that runs entirely in your browser — no installation, no account, no server required.

**Live site:** [https://peterlwkww-ai.github.io/Learning](https://peterlwkww-ai.github.io/Learning)

---

## What It Does

- Record daily expenses in seconds — just type what you spent on and the amount
- Auto-detects the category from your description (e.g. "coffee" → Food, "grab" → Transport)
- Browse expenses by day using arrow buttons or a calendar date picker
- See today's total and this month's total at a glance
- Visual breakdown of spending by category with progress bars
- All data is saved privately on your own device (browser localStorage)

## How to Use

1. Open the [live site](https://peterlwkww-ai.github.io/Learning) on your phone or computer
2. Type what you spent on in the description box
3. Enter the amount
4. Tap **+ Add Expense**

To browse a past day, use the **← →** arrows or tap the **📅 calendar** icon to jump to any date.
